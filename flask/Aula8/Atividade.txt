Nesta atividade você vai desenvolver um sistema web de consulta de especialidades médicas usando o framework Flask. 
O usuário digita o nome de uma especialidade (ex.: Cardiologia) e o sistema exibe os médicos disponíveis, seus CRMs e os planos de saúde que eles aceitam.


Objetivos
•	Criar e organizar um projeto Flask do zero
•	Entender o ciclo requisição → servidor → resposta
•	Trabalhar com rotas GET e POST
•	Acessar dados enviados por formulários HTML
•	Passar listas e dicionários Python para templates Jinja2
•	Tratar entradas inválidas e exibir mensagens de erro
